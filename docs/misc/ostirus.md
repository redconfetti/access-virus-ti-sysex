# OsTIrus

[OsTIrus](https://theusualsuspects.io/downloads/ostirus) is a multi-platform
audio plugin (VST3, AU, CLAP, LV2) that emulates the Freescale DSP56300
processors used in the Access Virus TI family. It runs original Virus TI
firmware and presents the TI UI.

- [OsTIrus downloads](https://theusualsuspects.io/downloads/ostirus)
- [Gearmulator / DSP56300 emulation](https://github.com/dsp56300/gearmulator)

The SysEx specifications in this repository may also apply when controlling
OsTIrus over MIDI.
