# Part Levels

These multis are the same as `sysex/multis/base/base.syx`, however they
are different in one aspect as noted.

* pt-1-lvl-0.syx - Part 1 level set to -64 (0)
* pt-1-lvl-127.syx - Part 1 level set to +63 (127)
* pt-2-lvl-0.syx - Part 2 level set to -64 (0)
* pt-2-lvl-127.syx -  Part 2 level set to +63 (127)
